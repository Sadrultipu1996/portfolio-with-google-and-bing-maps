Routing

An airline wishes to create a map on its website to show off new routes from Cardiff Airport. These go to Dublin, Edinburgh, Leeds, Bangor, Southampton, Stanstead and Exeter. Create a Google Map and use Polylines to present this information. Make it look nice, add a suitable title and some supportive text. Perhaps use different line styles to show the frequency of flights? Add information to your map to explain what any colours, line thicknesses, or other settings relate to. This is another assessed exercise.
